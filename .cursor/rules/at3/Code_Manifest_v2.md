# Appendix: Code Manifest (v2 – Exhaustive)

This appendix provides a comprehensive and exhaustive list of the **263** source and test files within the `src` directory of the MotionMingle project. It is offered in four logical chunks (60 + 60 + 60 + 83) to facilitate review.

---

## Guide to Contribution Terms

* **Architected and Implemented** – Led high-level design and wrote core implementation from scratch.
* **Designed and Created** – Planned structure and authored foundational asset (e.g. DB schema).
* **Authored and Implemented** – Primary author of tests, crafted strategy, assertions, and mocks.
* **Created and Implemented** – Built frontend features/hooks/components including styling & logic.
* **Configured and Integrated** – Integrated standard tools/boilerplate, ensured proper configuration.
* **Adapted from Component Library** – Started from shadcn/ui, customised & integrated into design system.
* **Generated and Utilised** – Auto-generated by tooling; contribution in configuring and consuming output.
